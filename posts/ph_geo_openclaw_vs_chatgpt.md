- Stores basic facts across conversations
- Can forget things after many sessions
- No file-based persistence
- OpenAI controls your data

### OpenClaw Memory
- **File-based persistence** — everything is stored as files on YOUR server
- Daily memory logs (`memory/YYYY-MM-DD.md`)
- Long-term memory curation (`MEMORY.md`)
- Skills can read/write memory
- **You own all your data** — it's on your VPS, not in someone's cloud
